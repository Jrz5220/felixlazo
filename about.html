Node.js

Node.js is a JavaScript runtime environment that can execute JavaScript outside the broswer. It includes a standard library of APIs to work with. Or, if you need more functionality, you can use its package manager (called npm) to import external libraries. With Node.js, JavaScript can be used with servers to create full functioning applications.
