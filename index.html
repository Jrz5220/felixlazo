ATM Simulator

A Java web app that simulates an ATM. Allows users to login with their account and pin numbers. If they don’t have a bank account, they can register for one. Once logged in, users can update their bank balances (checking/saving) and account settings. Includes client-side and server-side validation.

Backend:
Java, Maven, Spring MVC, Spring Security, Hibernate, AWS Elastic Beanstalk, AWS RDS (MySQL engine)
Frontend:
JSP, JSTL, CSS3, JavaScript, Bootstrap 5
